# XCX_Diary

### **简介**

这是我个人练习小程序的一篇代码，主要实现了日记的记录，标签选择，背景更换，字体颜色更换，轮播图，个人信息的展示，下拉选择框等功能。

基于ColorUI组件库完成了大部分的设计。
